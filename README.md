## ğŸ® Usage

### Creating Your First Creative

1. **Navigate to Creative Builder**
   - Click "Creative Builder" in sidebar
   - Choose background color or upload custom

2. **Add Product Images**
   - Drag & drop product packshots
   - Click to add to canvas
   - Auto-positioned and scaled

3. **Generate AI Background (Optional)**
   - Enter description: "festive Christmas scene"
   - Click "Generate AI Background"
   - Watch magic happen!

4. **Add Text**
   - Type promotional message
   - Click "Add Text to Canvas"
   - Position as needed

5. **Export All Formats**
   - Review compliance indicators (âœ“)
   - Click "Export All Formats"
   - Download 4 optimized files instantly

### Advanced Features

**Multi-Variant A/B Testing:**
```
1. Create base campaign
2. Navigate to Multi-Variants
3. Click "Generate 3 Variants"
4. Compare predicted CTR
5. Export best performer
```

**Voice Input:**
```
1. Go to Create Campaign
2. Click microphone icon
3. Speak campaign brief
4. AI transcribes automatically
```

**AI Chat Assistant:**
```
1. Navigate to AI Assistant
```
